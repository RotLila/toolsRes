\subsection{调整虚拟机窗口大小}%
\label{sec:vbox-tip-window-size}

至少有两种方法：
\begin{enumerate}
    \item 按组合键 Host + H.（组合键在 Linux 和 Windows 上通常为右边的
        Ctrl, 在 Mac 上通常为左边的 Command 键）
    \item 用鼠标拖动窗口，调节窗口大小。
\end{enumerate}
